About Quran Grammar

Learn Arabic grammar through Quranic text analysis and morphology

Disclaimer

This is an educational web application designed to help students learn Arabic grammar through Quranic text analysis. All content is fetched from third-party sources and we do not claim ownership of any religious texts or grammatical analyses.

Data Sources

This application fetches data from the following trusted sources:

E'arab (Grammatical Analysis)

  • Surah Quran - surahquran.com Provides detailed Arabic grammatical analysis (E'arab) for Quranic verses
  • Quran API - quranapi.pages.dev Supplies English and Urdu translations of Quranic verses

Sarf (Morphology Analysis)

  • AraTools - aratools.com Provides Arabic morphological analysis including root, form, and grammatical details

Features

E'arab Analysis
Detailed grammatical breakdown of Quranic verses with English and Urdu translations
Sarf Analysis
Morphological analysis of Arabic words including root derivation and grammatical forms
Mobile Responsive
Optimized for both desktop and mobile viewing
Privacy First
Pure client-side application with zero data collection

Privacy & Data

This application runs entirely in your browser and does not collect, store, or transmit any personal data. All requests are made directly to third-party APIs from your device. Please refer to the privacy policies of the source websites for information about how they handle requests.
